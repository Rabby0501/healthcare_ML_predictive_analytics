# Diabetes Prediction Dashboard (Streamlit)

A real-time medical decision support system using Machine Learning models:
- Decision Tree
- Random Forest
- Naive Bayes
- Logistic Regression
- XGBoost

## Features
- Live prediction
- Animated probability gauge
- PDF report generation
- Doctor recommendation system
- Model performance comparison
- Confusion matrix & ROC
- Data insights (heatmap, distribution plots)
